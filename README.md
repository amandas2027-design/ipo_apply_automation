# ipo_apply_automation
A Python automation script built using Playwright to automatically apply for IPOs on the MeroShare platform for multiple users.  This tool reads user data from a CSV file and performs end-to-end automation including login, IPO selection, form filling, and submission — all in headless (background) mode.
🚀 Features
🔐 Automated login for multiple users
📊 Bulk processing via CSV input
🔍 Smart company detection and Apply button handling
🏦 Auto-selection of bank and account
📝 Automatic IPO form filling (Kitta, CRN, PIN)
🖥️ Headless execution (runs in background)
📸 Debug screenshots for failed attempts
📊 Final summary report (success/failure per user)
